# Changelog

Все заметные изменения в проекте документируются здесь.

## [0.9.0] - 2026-03-05 — Большая переработка

### Изменено
- Полная переработка AI-моделей: переход на проверенные OpenRouter модели
- Бесплатные модели первыми в ротации, платные как запас
- Переработка промптов — краткие ответы без воды

### Добавлено
- Функция сбора задач из лога, требующих действий
- Функция отчёта выполненных задач/подзадач/комментов за период
- Расширенное меню кнопок в Telegram
- Полная документация проекта (CHANGELOG, TODO, ARCHITECTURE, ROADMAP)

### Исправлено
- Все модели OpenRouter заменены на реально существующие ID
- Убраны несуществующие модели: optimus-alpha, nemotron-nano-8b, gemini-2.5-flash-preview

## [0.8.0] - 2026-03-04 — OpenRouter

### Изменено
- Миграция с Gemini на OpenRouter (Gemini заблокирован из России)
- Ротация бесплатных моделей при 429

### Исправлено
- Groq → Gemini → OpenRouter (из-за блокировок по IP)

## [0.7.0] - 2026-03-03 — Docker deploy

### Добавлено
- Dockerfile, docker-compose.yml, .dockerignore
- Деплой на Coolify VPS
- Cron-контейнер для ai_prioritizer и knowledge_base_sync

## [0.6.0] — Telegram бот

### Добавлено
- Telegram бот с 4 кнопками: Транскрипт, Приоритизация, Задачи, Сбросить чат
- AI-чат "Вася" — пацанский ассистент
- Приоритизация задач через AI
- Аудио-транскрипция
