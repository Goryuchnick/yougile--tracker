# Фича: Транскрипт созвона → автораспределение задач

## Концепция

Аудио/текст созвона → AI транскрибирует → Извлекает задачи → Создаёт в YouGile

Для каждой задачи AI определяет:
- Название и описание с контекстом
- Исполнитель (маппинг имя → YouGile user ID)
- Дедлайн (абсолютный или вычисленный из относительного)
- Приоритет (стикер High/Medium/Low)
- Чеклист подшагов

## Варианты подачи транскрипта

### Вариант A: Аудиофайл в Telegram (Phase 1)
Пользователь скидывает запись → Gemini → список задач → подтверждение → YouGile.

### Вариант B: Текст транскрипта (Phase 1)
Команда `/meeting` + текст или `.txt` файл.

### Вариант C: Автоматика через tl;dv (Phase 2)
Вебхук `TranscriptReady` → сервер → Gemini → YouGile. Полностью hands-free.

## AI-промпт для извлечения задач

```
Ты — опытный проект-менеджер. Проанализируй транскрипт совещания
и извлеки ВСЕ задачи, поручения и обязательства.

Для каждой задачи определи:
- "title" — краткое название (до 80 символов)
- "description" — описание с контекстом из совещания
- "assignee" — кто ответственный (имя или "не назначен")
- "deadline" — дата YYYY-MM-DD если упомянута, иначе null
- "priority" — "High"/"Medium"/"Low"
- "checklist" — список подшагов, если упомянуты

Правила:
- Извлекай ТОЛЬКО действия: "я сделаю...", "нужно...", "давайте...", "[Имя], пожалуйста..."
- Сегодняшняя дата: {current_date}
- Если дедлайн относительный ("к пятнице") — вычисли дату

Верни JSON массив.
```

## План реализации

### Phase 1 (3-5 дней)
1. Хэндлер для аудиофайлов (.mp3, .m4a, .wav)
2. Многозадачный промпт извлечения
3. Шаг подтверждения (inline-кнопки)
4. Маппинг участников (имя → user ID)
5. Команда `/meeting` для текста

### Phase 2 (1-2 недели)
6. Вебхук tl;dv → автообработка
7. FastAPI-сервер для вебхуков

## Интеграция с Obsidian CLI (v1.12+)

Obsidian 1.12 выпустил CLI — управление хранилищем из терминала.

| Сценарий | Как использовать |
|----------|-----------------|
| YouGile → Obsidian | Завершённые задачи из `knowledge_base_sync.py` пушить как заметки в vault |
| Obsidian → YouGile | Заметки с тегом `#task` парсить и создавать задачи в YouGile |
| Транскрипт → Obsidian | AI-обработка созвона сохраняется и как заметка с линками между встречами |
| Дашборд в Bases | Таблица задач YouGile синхронизируется как Bases-файл в Obsidian |

**Установка Obsidian CLI:** `winget install Obsidian.Obsidian`, затем CLI доступен как `obsidian`.
