
YouGile Bot Service
===================

Управление ботом:

1.  **run_bot.bat** - Запустить бота вручную (с окном консоли).
2.  **run_bot_hidden.vbs** - Запустить бота скрыто (без окна).
3.  **stop_yougile_bot.bat** - Остановка бота (ВНИМАНИЕ: закрывает все python процессы).

Автозапуск:
Бот добавлен в автозагрузку Windows. Он будет запускаться автоматически при входе в систему.

Если бот перестал отвечать:
1. Запустите `stop_yougile_bot.bat`
2. Запустите `run_bot.bat` (чтобы видеть ошибки) или `run_bot_hidden.vbs`.
