# Деплой на Coolify

Деплой текущего состояния проекта на сервер через Coolify.

## Предварительные проверки
1. Проверь `git status` — нет ли незакоммиченных изменений
2. Проверь что `.env` НЕ попадёт в коммит (есть в `.gitignore`)
3. Проверь `requirements.txt` — все зависимости актуальны

## Процесс деплоя
1. `git add` нужные файлы (НЕ секреты)
2. `git commit` с описательным сообщением
3. `git push origin ${0:-main}` — пуш в ветку
4. Coolify подхватит изменения автоматически через webhook

## Конфигурация Coolify
- Сервер: 8GB RAM, 3 ядра
- На сервере уже 3 сайта — следи за потреблением ресурсов
- Приложение запускается через `python bot/yougile_bot.py`
- Переменные окружения задаются в панели Coolify (не в файлах)

## После деплоя
1. Проверь логи: `ssh server 'docker logs yougile-bot --tail 50'`
2. Убедись что бот отвечает в Telegram
3. Сообщи пользователю статус деплоя

## Важно
- НИКОГДА не пушь `.env`, `data/yougile_key.txt` или файлы с ключами
- Ветка `main` = продакшен, `dev` = разработка
- Перед пушем в main — проси подтверждение у пользователя
